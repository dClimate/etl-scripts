# Description
VHI stands for Vegetation Health Indices (VHI), and is one of the datasets provided by NESDIS STAR, an organzation under NOAA.
For more information, see [https://www.star.nesdis.noaa.gov/smcd/emb/vci/VH/index.php](https://www.star.nesdis.noaa.gov/smcd/emb/vci/VH/index.php)
